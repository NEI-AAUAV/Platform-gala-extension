# Gala Extension

Gala is the NEI Platform's event management extension for the annual Gala dinner. It handles table reservations, voting, and event administration.

## Structure

- `api-gala/` – FastAPI service (Python, MongoDB)
- `web-gala/` – React/Vite frontend (TypeScript, Tailwind)

## Status

This extension is currently being migrated from the Platform monorepo to a standalone repository. The code will be added in a subsequent commit.
